
# Thomson Theme Manager

**Thomson Theme Manager** — это расширение для браузеров на базе Chromium (Manifest V3), которое позволяет создавать, хранить и применять кастомные темы оформления для форума **Majestic RP** (движок XenForo).  
Вы можете менять внешний вид форума через CSS и добавлять динамические эффекты через JavaScript — всё в пару кликов.

---

## Основные возможности на данный момент

- **Создание тем** – задайте название, цвет/логотип, вставьте свой CSS и JS-код.
- **Библиотека тем** – все созданные темы хранятся локально в расширении.
- **Включение/выключение** – одним переключателем применяйте нужную тему на форуме (активная тема автоматически отключает остальные).
- **Редактирование** – правьте код темы в любой момент.
- **Экспорт/импорт** – скачайте тему в JSON или загрузите готовую, чтобы поделиться с друзьями.
- **Обход CSP** – JS-код выполняется через `chrome.scripting.executeScript` в главном мире, поэтому работают даже сложные скрипты (например, анимированные частицы).
- **Автоматическая подмена футера** – внизу форума отображается `Redesigned by Thomson`.

## Установка

1. Скачайте или клонируйте репозиторий:
   ```bash
   git clone https://github.com/yourname/thomson-themes.git
   ```
2. Откройте браузер Chrome (или Edge) и перейдите на страницу расширений:
   - `chrome://extensions/` (или `edge://extensions/`)
3. Включите **«Режим разработчика»** (Developer mode).
4. Нажмите **«Загрузить распакованное расширение»** и выберите папку с расширением.
5. Готово! Иконка расширения появится на панели браузера.

## Как пользоваться

После установки нажмите на иконку расширения, чтобы открыть попап-менеджер.

### Вкладка «Главное»
- Краткая информация о расширении.
- Кнопки для **экспорта** всех тем в JSON и **импорта** сохранённых тем.

### Вкладка «Мои темы»
- Список всех созданных тем (включая дефолтную).
- Для каждой темы:
  - **Переключатель** (вкл/выкл) — при включении страница форума автоматически перезагружается, чтобы применить новую тему.
  - **Редактировать** — открывает форму создания с предзаполненными данными.
  - **Скачать** — экспортирует только эту тему в JSON.
  - **Удалить** (кроме дефолтной).

### Вкладка «Создать»
- Заполните:
  - **Название темы** (обязательно).
  - **Логотип** — выберите цвет или загрузите картинку (отображается в списке).
  - **CSS-код** – любые стили для форума (можно использовать `!important` для переопределения).
  - **JavaScript-код** – например, скрипт для добавления анимированных звёзд, изменения фона и т.п.
- Нажмите **«Сохранить тему»** — она сразу включится и станет активной.

## Пример: тема «Тёмное небо» с партиклами

```css
/* Тёмный фон и закругления */
body { background: #0b0b12; color: #ddd; }
.p-body-inner { background: rgba(18,18,30,0.85); backdrop-filter: blur(12px); border-radius: 32px; }
/* ... остальные стили ... */
```

```javascript
// Создаём 250 звёзд, которые дрейфуют и мерцают
(function() {
  const container = document.createElement('div');
  container.id = 'thomson-particles';
  container.style.cssText = 'position:fixed;top:0;left:0;width:100%;height:100%;pointer-events:none;z-index:0;overflow:hidden;';
  document.body.prepend(container);
  // ... код создания звёзд
})();
```

Больше готовых конфигов и примеров можно найти в репозитории в папке `examples/`.

## Технические детали

- **Manifest V3** – полная совместимость с современными браузерами.
- **Хранилище** – все данные сохраняются в `chrome.storage.local`, никаких внешних запросов.
- **Безопасность** – расширение работает только на домене `forum.majestic-rp.ru`, данные никуда не уходят.
- **CSP** – обход через `chrome.scripting` с `world: 'MAIN'`, поэтому JS-код тем выполняется без ограничений.
