# Thomson Theme Manager

**Thomson Theme Manager** — это расширение для браузеров на базе Chromium (Manifest V3), которое позволяет создавать, хранить и применять кастомные темы оформления для форума **Majestic RP** (движок XenForo).  
С его помощью вы можете полностью изменить внешний вид форума — от цветовой схемы и шрифтов до сложных анимаций — без необходимости редактировать код страницы вручную.

---

## Основные возможности

- **Создание тем**  
  Задайте название, выберите логотип (цвет или изображение), напишите CSS и (опционально) JavaScript-код. Тема сразу готова к использованию.

- **Встроенная библиотека тем**  
  В новой вкладке «Библиотека» доступны темы, созданные сообществом и опубликованные в официальном репозитории.  
  - Поиск по названию, автору или описанию.  
  - Фильтрация по категориям (Тёмная, Светлая, Космос, Киберпанк и др.).  
  - Быстрая установка и обновление тем прямо из интерфейса.

- **Обновление тем**  
  Если установленная тема имеет более новую версию в библиотеке, появится кнопка «Обновить» — нажмите её, чтобы получить актуальную версию без потери настроек.

- **Управление темами**  
  Все созданные или установленные темы хранятся в разделе «Мои темы»:  
  - Включайте/выключайте тему одним переключателем (активная тема автоматически отключает остальные).  
  - Редактируйте код, название или логотип в любой момент.  
  - Удаляйте ненужные темы (кроме стандартной).  
  - Скачивайте любую тему в формате JSON, чтобы поделиться с друзьями.

- **Удобный поиск и сортировка**  
  В «Моих темах» доступен поиск по названию и сортировка по дате или имени.

- **Предпросмотр**  
  Перед активацией можно открыть модальное окно с визуализацией темы — увидеть, как она будет выглядеть на форуме.

- **Безопасный режим**  
  При включении этого режима JavaScript-код из **недефолтных** тем не выполняется — применяется только CSS. Это защищает вас от потенциально опасных скриптов в темах из непроверенных источников.  
  Для стандартной (дефолтной) темы JS выполняется всегда, так как она считается надёжной.

- **Экспорт/импорт**  
  - Экспортируйте все свои темы в один JSON-файл для резервного копирования.  
  - Импортируйте темы из JSON — при этом отображается предупреждение о возможных рисках, а также доступна опция «Импортировать только CSS» для дополнительной безопасности.

- **Обход CSP**  
  JavaScript-код тем выполняется через `chrome.scripting.executeScript` в главном мире страницы, поэтому работают даже сложные скрипты (например, анимированные частицы, динамические фоны).

## Установка

1. Скачайте или клонируйте репозиторий:
   ```bash
   git clone https://github.com/Thomson-Hate/Thomson-Theme-Manager---Majestic-RP-Forum.git
   ```
2. Откройте браузер Chrome (или Edge) и перейдите на страницу расширений:
   - `chrome://extensions/` (или `edge://extensions/`)
3. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу.
4. Нажмите **«Загрузить распакованное расширение»** и выберите папку с расширением.
5. Готово! Иконка расширения появится на панели браузера.

## Как пользоваться

После установки нажмите на иконку расширения, чтобы открыть попап-менеджер.

### Вкладка «Главное»
- Краткая информация о расширении.
- Кнопки для **экспорта** всех тем в JSON и **импорта** сохранённых тем (с предупреждением о безопасности).
- Переключатель **«Безопасный режим»** — включите его, если не доверяете импортируемым темам.

### Вкладка «Мои темы»
- Список всех ваших тем (включая стандартную).
- Для каждой темы:
  - **Переключатель** — включение/выключение (страница форума перезагружается автоматически).
  - **Предпросмотр** — посмотреть, как тема выглядит.
  - **Редактировать** — изменить код или параметры.
  - **Скачать** — экспортировать тему в JSON.
  - **Удалить** (кроме дефолтной).
- Поиск по названию и сортировка.

### Вкладка «Создать»
- Заполните поля:
  - **Название темы** (обязательно).
  - **Категория** — для удобной фильтрации в библиотеке.
  - **Описание** (необязательно).
  - **Логотип** — выберите цвет или загрузите картинку (отображается в списке).
  - **CSS-код** – любые стили для форума.
  - **JavaScript-код** – скрипты для анимаций, динамических эффектов и т.п.
- Нажмите **«Сохранить тему»** — она сразу включится и станет активной.

### Вкладка «Библиотека»
- Здесь отображаются темы из официального репозитория.
- Используйте поиск и фильтр по категории, чтобы найти нужную.
- Для каждой темы:
  - **Предпросмотр** — оценить внешний вид.
  - **Добавить** — установить тему (она появится в «Моих темах» и активируется).
  - **Обновить** — если у установленной темы есть новая версия, кнопка заменится на «Обновить».
- Библиотека автоматически обновляется раз в час, но вы можете нажать на кнопку **🔄** для ручного обновления.

## Пример: тема «Тёмное небо» с партиклами

**CSS** (фрагмент):
```css
body { background: #0b0b12; color: #ddd; }
.p-body-inner { background: rgba(18,18,30,0.85); backdrop-filter: blur(12px); border-radius: 32px; }
/* ... остальные стили ... */
```

**JavaScript** (фрагмент):
```javascript
(function() {
  const container = document.createElement('div');
  container.id = 'thomson-particles';
  container.style.cssText = 'position:fixed;top:0;left:0;width:100%;height:100%;pointer-events:none;z-index:0;overflow:hidden;';
  document.body.prepend(container);
  // создание 250 звёзд с анимацией
})();
```

Больше готовых примеров можно найти в папке [`examples/`](examples/) репозитория.

## Технические детали

- **Manifest V3** — полная совместимость с современными браузерами.
- **Хранилище** — все данные (темы, настройки, кэш библиотеки) сохраняются в `chrome.storage.local`. Никаких внешних запросов, кроме загрузки библиотеки с GitHub.
- **Безопасность** — расширение работает только на домене `forum.majestic-rp.ru`, данные никуда не передаются.
- **Обход CSP** — через `chrome.scripting` с `world: 'MAIN'`, поэтому JS-код тем выполняется без ограничений контент-политики.
- **Автообновление библиотеки** — раз в час, чтобы вы всегда видели актуальные темы.

## Безопасность

Мы уделяем большое внимание безопасности:

- **Безопасный режим** — блокирует выполнение JS в темах, кроме дефолтной.
- **Предупреждение при импорте** — перед загрузкой JSON показывается диалог с описанием рисков.
- **Опция «Импортировать только CSS»** — при импорте можно исключить JS-код, оставив только стили.
- Все темы проверяются на наличие обязательных полей, отсутствующие поля заполняются безопасными значениями.

Рекомендуем всегда включать **Безопасный режим**, если вы импортируете темы из непроверенных источников.
